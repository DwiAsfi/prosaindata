  2. Login dengan akun yang sudah terdaftar di orchest

  3. Setelah itu pada halaman orchest cloud instances pilih busy-euler (free) dan klik open

  4. Buat projek dengan nama prosaindata kemudian buat new pipeline dengan nama getdata.ipynb

  5. Klik pipeline isi step name kemudian klik edit in jupyterlab

  6. Kemudian masuk ke Environment dan masukkan kode dan klik build tunggu hingga proses build sukses
